四、页面布局时常见的错误
	①容器中的第一个孩子，如果声明了margin-top，会影响容器——margin-top溢出
	②容器中的最后一个孩子，如果声明了margin-bottom，会影响容器本身——margin-bottom溢出
	解决办法：使用内容生成，创建新的table元素，成为第一个/最后一个孩子
	.box:before,  .box:after {
		content: '';
		display: table;
	}

五、CSS3中的内容生成
  通常情况下，一个元素中的内容都是使用HTML编写的；但是有些场合下也还可以使用CSS向元素的开始和结束部分添加一些内容——称为“前置内容生成”和“后置内容生成”
  选择器:before {
	content: '生成的前置内容'
  }
  选择器:after {
	content: '生成的后置内容'
  }
 
 
六、响应式网页必需知识点：CSS3 Media Query
  媒体：指浏览网页的设备，如screen、print、speech等；
  媒体查询：查询浏览网页设备的特性，例如：屏幕的宽度/高度/解析度、打印机是彩色/单色....根据查询得到的结果执行某些代码，而不执行其它的代码。
